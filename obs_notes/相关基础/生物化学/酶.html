Bio-catalyst（生物催化剂）

反应物 => 底物（substrate）；产物依旧是产物（products）

化学本质：all protein => protein + RNA(ribozyme（核酶）) <--> 核酸酶（水解核酸）

RNA-based enzyme：

  • 发现
  • 实例
    • 催化肽键形成（All cell）
    • 核糖核酸酶P（protein+RNA） => 剪切多余RNA序列

{部分关于酶的高中知识}（更快；存在 saturation；条件性更苛刻 & 专一性更高）

特殊：

  • 酶正负反应均催化（催化剂也是如此）

特有性质：

  • 效率极高
    • 碳酸酐酶（催化碳酸）
    • 大幅降低活化能（过渡态理论 <=> 和中间物不一样）
  • 必须和底物结合
    • 活性中心 area 底物 -> 产物
      • 体积占比很小
      • 3D结构
      • 残基：催化作用 + 结合 => 亲水
    • 专一性
      • 一种/一类/立体
        • 琥珀酸脱氢酶的产物是反式的
      • 「🔒-🔑」模型 Fisher => stable
      • 诱导契合模型 Koshland（也提出了 蛋白质 > Examples 的序列模型） => flexible
        • 实例：己糖激酶（glucose -> 6-P-glucose, Mg+ required）=> 结合底物后构象变化，蛋白质中心的水分子被移走（反应疏水，一旦有水将导致副反应： ATP 水解）；1st step in 三羧酸循环（两次，需要水分子）
        • 底物诱导，才会契合
      • 三点附着模型——≥3个点
        • 实例：2nd step in 三羧酸循环（柠檬酸（手性） -> 异柠檬酸；羟基转移到来源于草酰乙酸而非乙酰 CoA 的碳上）
    • 潜力极大
    • 对外界条件敏感
      • 最适条件不是常数

组成：

变构酶

变构位置/中心 + (变构效应/调节物 -> 不变) => 诱导酶的变构 => 改变活性

  • 变构激活剂
  • 变构抑制剂

性质：

  • 催化可调控 => 一般是不可逆反应
    • ATP as a 变构效应物 to 调节 process: 糖酵解
  • 非共价键结合
    • 结合是可逆的
  • 这种酶大多具备四级结构
    • ecp 大肽链的（很少）
  • 动力学 S-shape 曲线（大多数） <== 正协同效应
  • 变构酶也有竞争抑制（S -> non-S => 低浓度下会产生激活的效果）
    • 一个被抑制，其他基团被激活
    • 变构抑制剂：more T-state 正协同效应++
    • 变构激活剂： 正协同效应-- 甚至双曲线
    • T/R-state

动力学特点与竞争抑制

S 型曲线的来源在于正协同效应（参见血红蛋白与肌红蛋白），结合后会使酶发生构象变化进而改变亲和性。

不同的抑制剂：

  • 竞争性抑制剂在低浓度时结合酶能够诱导变构使其他基团处于亲和的状态。
  • 在高浓度下回和抑制剂结合更多 => 抑制
  • 别构抑制剂 => S 曲线后移 && 更平缓 -> 更「S 型曲线」
  • 别构激活剂 => S 曲线前移 && 更急促 -> 双曲线
    • T -> R

过渡态稳定学说

酶的活性中心 ~ 过渡态（过渡态类似物）类比：弯折铁丝使其断裂——过渡态：铁丝弯折部发热

  • case 酶适合结合底物 => 结合时反而能量最低，难以反应；
  • case 酶适合结合过渡态 => 过渡态时能量不会很高，易于产生产物

证据

  • 过渡态类似物 <--> 亲和力及其高（通常是不可逆）
    • L-Pro -过渡的平面态-> D-Pro => 设计药物
  • abzyme（抗体酶）：可催化反应的抗体
    • 过渡态类似物 as Antigen -> combine
    • cocaine 成瘾 => 设计催化水解的酶

稳定的手段

常见：

  • 定向与邻近效应
    • 双底物效应
    • 「酶：分子媒婆」——“撮合”分子使其在特定的位置以及角度 => 有效碰撞
    • 分子内的反应更容易发生
  • 广义酸碱催化（酸/碱）（讲了很多）
    • 获得/失去 proton（酶的活性反应中心的氨基酸残基；失去 Proton 为酸）
    • 等 pH 下的浓度变化 => 反应有变化（特异性不会发生变化）
    • RNAse A：酸/碱催化均有
  • 金属催化
    • 正负电荷 => 中和 => 稳定过渡态
    • 金属离子
      • Lewis 酸碱（单相离子平衡 > 酸碱概念）
      • 与带负电荷的底物结合，屏蔽负电荷，促进底物在反应中正确定向
      • 参与静电催化，稳定过渡态
      • 参与氧化还原，电子传递链 —— CystC
      • 稳定三维结构
  • 底物形变
    • 化学键更易断

共价催化：

  • 酶 --> 形成共价键（可断裂）
  • 产生了亲核（prefer）/亲电反应（多次）
    • --> 羰基 => 共价

实例

  • 蛋白酶（使蛋白质水解的酶）
    • 丝氨酸蛋白酶 CovB
      • 胰蛋白酶/凝乳蛋白酶/凝血因子/枯草杆菌蛋白酶
      • 有共同祖先 =>
      • Ser + His + Asp => 催化三元体
      • 稳定过渡态的机制
        • 氧阴离子洞
        • 催化三元体
          • Asp --- His --- Ser
      • 对氨基酸（提供羧基 => 肽键）有要求
    • 金属蛋白酶（需要金属离子）
      • <== 金属螯合剂
    • 羧基蛋白酶/天冬氨酸蛋白酶
      • 艾滋病病毒
    • 巯基蛋白酶/半胱氨酸蛋白酶 CovB
    • 共有：碳四面体过渡态（亲核进攻 => 肽键的羰基）1~2 次亲核进攻

相同性与不同性

同：

  • 催化效率都很高
  • co-factor 主要是金属离子
  • 有活性中心
  • 过渡态稳定机制依旧有效
  • 折叠时依旧有差别

异：

  • 金属离子起重要作用
  • 仅催化少数反应
    • 类型有限
    • 效率不如蛋白质
    • e.g. 转酯反应
