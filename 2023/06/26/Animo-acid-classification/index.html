氨基酸

通论

哪怕 R 集团（Residual group）憎水，所对应的氨基酸都溶于水。

以下内容从去年的生化笔记中照搬，后期可能会有更改。

化学性质
  • capacity to polymerize
    • peptide => protein
    • 氨基 + 羧基 –> 肽键/酰胺
  • chirality
    • C* between -NH3 and -COOH (except G)
    • L-Amino Acid <–> L-gly…（甘油醛）
      • L/D 分类
        • 上 -C=O; 右 -H; 左 -(>C e.g. O/N)
        • L-type mainly
  • novel acid-base properties
    • zwitterions（两性离子）
      • AA –> +AA- （两性离子；共性）
    • pI
      • pH(NetCharge = 0)
      • Constant
      • 计算
      • 实验测定
  • varties structure and chemical functionality
    • 部分氨基酸的氨基 + 亚硝酸 ==> 氮气（凯氏定氮法），expect Proline（属于亚氨基酸）
    • 茚三酮反应（含有氨基以及羧基的化合物 + 茚三酮 ==> 蓝紫色化合物），定量 & 定性，Proline ==> 黄色
    • DNFB（2， 4-二硝基氟苯）反应 ==> DNP-AA，可区分游离氨基酸；当蛋白质时，仅修饰氨基端
      • 确定蛋白质的 N-terminal Amino Acid
    • PITC反应 ==> 可使 N 段氨基酸游离 ==> 新的氨基被暴露
      • 与 NDFB 联合和确定氨基酸序列
    • R-group
      • 芳香族（Phe, Try, Trp）—— UV 280 nm 下有吸收峰
        • Trp > Try > Phe
        • 碱基（核酸）—— UV 260 nm
      • 羟基（Thr, Trp, Ser）
        • 磷酸化 ==> 结构变化 ==> 功能改变
        • 特定情况下会释放质子（Lewis acid）
      • 碱性氨基酸（esp. His）
        • 咪唑基（pKa 6.x 接近生理环境）

各论

21st/22nd PAA：==> 遗传密码子的改变（占位原来的终止子）

  • 硒代半胱氨酸（Sec, U）
    • 少数蛋白质 e.g. 去碘酶（T4 -> T3）
    • 某些酶： if 有硒 ==> 编码 else 终止
  • 吡咯赖氨酸（Pyl, O）
    • 更少见 产甲烷的细菌

20种常见的氨基酸：

  • hydrophobic（疏水的）
    • aliphatic
    • aromatic（芳香族）
    • sulfur（硫）-containing
  • hydrophilic（亲水的）
    • polar/uncharged
    • basic or acidic

Non-polar, hydrophobic：

  • Gly 甘 H
  • Ala 丙 甲基
  • Pro 脯 -CH2-CH2-CH2-
  • Val 缬 异丙基
  • Leu 亮 丁基
  • Ile 异亮 也是丁基

Non-polar, hydrophilic：

  • Ser 丝 羟甲基
  • Thr 苏 羟丁基（类似于苏糖酸）
  • Gln, Q 谷酰胺 氨酰[^1]-亚甲基
  • Asn, N 天冬酰胺 氨酰-1, 2-亚乙基
  • Met 蛋 CH3-S-1, 2-亚乙基-
  • Cys 半胱 HS-亚甲基-

[^1]: 氨酰 = 氨基酸 - 羟基

Aromatic, hydrophobic：

  • Phe, F 苯丙 （苯甲基 / 加上另外两个碳变成苯丙酸）
  • Trp, W 色 β-吲哚基丙氨酸，和胰岛素关系密切
  • Tyr, Y 酪 4-羟基苯丙氨酸，♥来自奶酪

Polar, charged：（有点酸碱；在生理环境下带电性）

  • Asp, D 天冬 acidic
  • Glu, E 谷 acidic
  • Sec, U Se-Cys acidic
  • Arg, R 精 basic
  • Lys, K 赖 basic
  • His 组 basic
