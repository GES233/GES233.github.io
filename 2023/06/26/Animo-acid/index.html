氨基酸

以下内容从去年的生化笔记中照搬，后期可能会有更改。

总论

全都溶于水（因为氨基以及羧基），哪怕 R 基憎水

结构特点

每一个分子都有一个氨基（anime）、一个羧基（carboxylic group）的化合物。

化学性质
  • 能够聚合（capacity to polymerize）
    • peptide => protein
    • 氨基与羧基在特定的情况下会脱水缩合形成肽键/酰胺
  • 手性（chirality）
    • 手性碳（C*）Carbon between -NH3 and -COOH (except G) => 存在镜像异构体
    • L-Amino Acid <--> L-glyceral（甘油醛）
      • L/D 分类系统（分类简单，与对应异构的甘油醛相比较即可）
        • L-... 的特征（按照 Fisher 投影式：横前竖后）：上有羰基（甘油醛是醛基，氨基酸是羧基）; 右是氢原子; 左是原子标号大于碳的原子（甘油醛是羟基（氧原子与碳相连）；氨基酸是氨基（氮原子与碳相连））
        • 生物体内主要有 L-type AA
          • 如果是通过 ribosome 合成 => 没有 D 型
          • 核糖体外的酶，没有 mRNA => 可能有 D 型
          • 核心：有没有和 tRNA 结合成对应的氨酰-tRNA（参与组合的酶有立体专一性）
  • 酸碱解离属性（novel acid-base properties）
    • zwitterions（两性离子）
      • AA --> ㊀A-A㊉ （两性离子；共性）
    • 等电点（pI）
      • pH(NetCharge = 0 <= Num(+) == Num(-))
      • Constant
      • 计算
      • 实验测定
  • 特异性结构与化学功能（varties structure and chemical functionality）
    • 部分氨基酸的氨基 + 亚硝酸 ==> 氮气（凯氏定氮法），expect Proline（属于亚氨基酸）
    • 茚三酮反应（含有氨基以及羧基的化合物 + 茚三酮 ==> 蓝紫色化合物），定量 & 定性，Proline ==> 黄色
    • DNFB（2， 4-二硝基氟苯）反应 ==> DNP-AA，可区分游离氨基酸；当蛋白质时，仅修饰氨基端
      • 确定蛋白质的 N-terminal Amino Acid
    • PITC 反应 ==> 可使 N 段氨基酸游离 ==> 新的氨基被暴露
      • 与 NDFB 联合和确定氨基酸序列
    • 不同的 R-group 带来的特点
      • 芳香族（Phe, Try, Trp）—— UV 280 nm 下有吸收峰
        • Trp > Try > Phe
        • 碱基（核酸）—— UV 260 nm
      • 羟基（Thr, Trp, Ser）
        • 磷酸化 ==> 结构变化 ==> 功能改变
        • 特定情况下会释放质子（Lewis acid）
      • 碱性氨基酸（esp. His）
        • 咪唑基（pKa 6.x 接近生理环境） ==> 蛋白质的活性中心常见
      • 2 * 半胱氨酸 => 胱氨酸（二硫键）

各论

蛋白质/非蛋白质AA

针对 22 种蛋白质氨基酸，分类依据，能否和 tRNA 结合。

非蛋白氨基酸（部分）：

  • 鸟氨酸（尿素循环）
  • 瓜氨酸（西瓜内含量丰富）
依据 side chain（R集团）分类

21st/22nd Protein AA：==> 遗传密码子的改变（占位原来的终止子）

  • 硒代半胱氨酸（Sec, U）
    • 和半胱氨酸很相似（S <- Se）
    • 少数蛋白质 e.g. 去碘酶中（实现 T4 -> T3）
    • 谷胱甘肽（一种三肽，有 Ox/Red 两种形式）过氧化物酶含有 Sec
    • 因为易氧化所以很少有游离态，前体为丝氨酸（Ser）
  • 吡咯赖氨酸（Pyl, O）
    • 更少见（动物没有！！！），常见于产甲烷的细菌/古菌
  • 某些酶在编码过程中：when 遇到终止密码子中的二者 if 有硒/吡咯 => 编码 else 终止

20种常见的氨基酸的分类：

  • hydrophobic（疏水的）
    • aliphatic
    • aromatic（芳香族）
    • sulfur（硫）-containing
  • hydrophilic（亲水的）
    • polar/uncharged
    • basic or acidic
Non-polar, hydrophobic
  • Gly/G 甘 H
  • Ala/A 丙 甲基
  • Pro/P 脯 -CH2-CH2-CH2-
  • Val/V 缬 异丙基
  • Leu/L 亮 丁基
  • Ile/I 异亮 也是丁基
Non-polar, hydrophilic

存在亲水基团但是不带电荷。

  • Ser/ 丝 羟甲基
  • Thr/ 苏 羟丁基（类似于苏糖酸）
  • Gln/Q 谷酰胺 氨酰1-亚甲基
  • Asn/N 天冬酰胺 氨酰-1, 2-亚乙基
  • Met/M 蛋 CH3-S-1, 2-亚乙基-
  • Cys/C 半胱 HS-亚甲基-
Aromatic, hydrophobic

带苯环。

  • Phe, F 苯丙 （苯甲基 / 加上另外两个碳变成苯丙酸）
  • Trp, W 色 β-吲哚基丙氨酸，和胰岛素关系密切
  • Tyr, Y 酪 4-羟基苯丙氨酸，♥来自奶酪
Polar, charged

有点酸碱；在生理环境下带电性。

  • Asp, D 天冬 acidic
  • Glu, E 谷 acidic
  • Sec, U Se-Cys acidic
  • Arg, R 精 basic
  • Lys, K 赖 basic
  • His 组 basic
依据营养价值

针对 20 种蛋白质氨基酸

  • 必需氨基酸（EAA）9种
    • from food
    • 『有一（异，I）组（H）笨（苯丙，F）蛋（M）精（R）来（赖，K）宿（苏，T）舍（色，W），喜欢晾（亮，L）凉鞋』
  • 半必需氨基酸（semi-EAA）2种
    • 『尤其是光（半胱，C）着腚吃奶酪（Y）』
    • e.g. Phe -引入羟基-> Try ~ PKU（苯丙酮尿症）
    • 小孩子/孕妇是必需的
