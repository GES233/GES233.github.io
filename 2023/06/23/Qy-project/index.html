Qy 计划的介绍

2023-06-23
|
  • 介绍

Qy 是从 18 至 19 年某几次头脑风暴催生出的的计划，即通过某种仿生的方式来使角色生成活灵活现的动作（包括但不限于声音、面部表情或身体上的动作）。

最原始的想法是想通过某种方法来尽可能的实现一个能够覆盖人类发声器官所能出现的绝大多数的声音的模型。

人话：做个会动会唱歌有表情的小姐姐。

早期想法

之前在看语音合成的相关资料的时候，想出了一个通过重现发声而不是对音频进行处理来生成音频的思路，并且试图再深入想了一下，但是因为对发声同时其他数据的采集的难度而暂时放弃。然后，今天刷知乎的时候，发现有人已经按照这个思路搞出来并且把论文发到 Nature 了，而且比我的想法更进一步：通过对发声相关的运动皮质的信号的采集，通过两个 RNN 模型（声道参数为其中间参量，其生成使用了其他大量的数据库）生成语音。生成的音质及准确率较以往的其他模型要好。

来自笔者的某条说说 2019年4月27日 22:29

最早的版本，仅仅局限于提出一种新的语音合成的思路罢了。按照原理来讲，它具有跨语言以及表达丰富且细腻的感情的潜能。

在高中某天参加社团百无聊赖的时候，我忽然意识到这种中间状态的思路可以有很丰富的表现力。就比方说：你完全可以从话语变成语音以及此刻发声器官的样貌、舌头的位置、胸廓的张开程度等等方面的内容。换言之，其会拥有更加细腻的表达能力。因此这个想法就一直没有丢掉。

关于名字

分成两部分说吧。Startwith Q and endaith y.

前者主要是受别的老哥的影响。刷油管看到个 Logisim CPU（用数字电路模拟软件整的），有个老哥用这玩意儿做演示创建文件的时候就说： This is a text file called "Q".

后者是不想要太过张扬的名字像整个闭口呼1的发音。就敲定了 i 或 y 。

Qi 已经有了，大概是「气」的意思。那就另一个了。

怎么念呢？

别人怎么念我不会管太多，没什么是必须的。可以拼出来读，也可以按照qui-/ki-（音标[kʰwɪ]/[kʷɪ]/[kɪ]）读，或者可以按照汉语拼音的 qí（注音［ㄑㄧˊ］）来念。

核心思路

用自然的原理来重现某些生理特征的思路在结果上是自然的，方法本身也具有可实现性。

说白了就是尽量细腻尽量精确的仿生能够产生非常自然细腻精细的动作。

四大支柱

  • 就旁观者而言，动作的产生是由施加动作的主体与环境的互动所产生的结果
    • 主体给予的是要执行动作时的「指令」，例如一系列相关运动神经元的放电
    • 以「文本转语音」（TTS）为例，主体与环境的互动指的是声带与气流的作用
  • 其中主体与环境的互动的部分机制难以完全复现
    • 相关算法的研究进展以及数据采集的难度，限制了在计算机上完全地仿真主体与环境交互的能力
  • 动作的被施加存在意识可控和不可控的成分
    • 意识无法极其精确地控制运动，意识也难以使主体完成其能力之外的动作，有道是「人力有穷尽，力所不能及」
  • 需要施加动作主体的当前状态作为感觉
    • 对应着上文的「互动」

人话版本

当时整出来这么一堆很奇怪的内容，已经让 ChatGPT 翻译过的还是很难懂的。

动作的产生是由施加动作的主体与环境的互动所产生的结果，

其中部分机制难以完全复现，而动作的施加则有可控和不可控的成分，

需要施加动作主体的当前状态作为感觉。

from ChatGPT

所以再在后面尝试用大白话逼叨逼叨。

举个例子，比方说一个正在打篮球的鸡，啊呸，美男子（以下简称鸡哥）。

我们希望鸡哥能够完成高难度的唱、跳、rap、篮球。

那么我们看到的铁山靠 捂裆派 美妙的舞蹈就是来源于鸡哥在主观意识下对身体的控制与环境互动（包括但不限于重力、空气阻力、作为生物而被限制着的身体机能等等方面）的产物。

SynapticStrings 是啥？

原来叫 Generic-Qy ，换了这个更酷的名字。

整这出是因为笔者在这个夏天吃了不少虚拟歌姬圈子的瓜（说句难听点的，小圈子逼事儿真多）。所以选择将原来的 Qy 拆分成两个内容，Qy 继续面向 ACGN 偏应用；另一个是纯粹的技术（针对人类动物的那种），与 Qy 受众不同，可能比较「枯燥」（说实话，两拨人本来就尿不到一壶去，能和睦相处的真不多，与其聚一块儿养蛊还不如分开）。

计划通过动物模型来构建原型以测试整个思路的可行性。
