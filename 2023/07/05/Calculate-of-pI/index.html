氨基酸等电点的计算

当只有氨基以及羧基时，粒子净电荷为零时的两边 或 取平均即可。

侧链含有可解离的基团的情况，以半胱氨酸为例：

假定放在酸性溶液中（质子化，正电荷），逐渐加碱直至净电荷为零。

将两侧反应的 取平均即可。

实例

赖氨酸（Lys）：

  • 9.0
  • 2.2
  • 10.5

酸性下：

  • => 净电荷 +1
  • => 净电荷 0
  • => 净电荷 -1
  • => 净电荷 -2

取两侧 平均数，可得：

酪氨酸（Tyr）：

  • 9.1
  • 2.2
  • 10.5

酸性下：

  • => 净电荷 +2
  • => 净电荷 +1
  • => 净电荷 0
  • => 净电荷 -2

计算过程同理。
